
# Movie Search App

This is a simple and interactive *Movie & Series Search Web App* built using *HTML, CSS, and JavaScript*.
It uses the [OMDb API](https://www.omdbapi.com/) to fetch real-time movie and series details.


# Features

* Search for any *movie or series* by name
* View *posters, titles, and types* instantly
* Filter results by *Movies*, *Series*, or *All*
* Handles empty or incorrect searches smoothly
* 100% built with *Vanilla JavaScript* — no external libraries

#  Technologies Used

 *HTML5* – for structure
 *CSS3* – for styling
 *JavaScript (ES6)* – for logic and interactivity
 *OMDb API* – for fetching movie data

#  How It Works

1. Type a movie or series name into the search bar
2. Click the *Search* button
3. The app fetches data from the OMDb API and displays results as cards
4. Use the dropdown menu to filter between *Movies*, *Series*, or *All*

# How to Run the Project

1. Clone this repository
   git clone https://github.com/Manasa-B-M/UI-Replica.git

2. Go into the project folder
   cd UI-Replica

3. Open the file *movie_task.html* in your browser
   That’s it — your app will start running locally!
